# Archive

This folder keeps historical documentation that is no longer part of the compact, active docs set.

## Archived source groups
- Old SDD specs and task documents from [../../specs](../../specs).
- Old capture and debugging notes from [../../apps/desktop/public](../../apps/desktop/public).
- Older working notes and investigation files from [../../research](../../research).

## Intent
- Preserve history without keeping it in the main documentation path.
- Avoid reintroducing SDD as the default documentation model.
- Keep the active docs set short and operational.
