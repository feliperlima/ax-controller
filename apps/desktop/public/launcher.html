AX WebSocket Capture Launcher

Método 1: Criar Bookmarklet (Recomendado)

Crie um bookmark que ativa a captura com um clique:

  1. Abra http://192.168.1.75/
  2. Crie um novo bookmark (Ctrl+Shift+B → Bookmark Manager)
  3. Cole este URL no campo "URL":
javascript:(function(){const s=document.createElement('script');s.src='http://192.168.1.75:8000/ax-capture-inject.js';document.head.appendChild(s);})();
  1. Salve com nome "AX Capture"
  2. Quando quiser capturar: clique no bookmark

Método 2: Console Manual

Cole o script diretamente no console do DevTools:

  1. Abra http://192.168.1.75/
  2. Pressione F12 para abrir DevTools
  3. Vá para aba "Console"
  4. Copie o código abaixo e cole no console:

Depois de Injetar - Teste

Teste A: CH1 com Microfone

AX_CAPTURE.clear() // Fale no microfone por 5 segundos AX_CAPTURE.summary() AX_CAPTURE.export()

Teste B: CH31/32 com Spotify

AX_CAPTURE.clear() // Roteie Spotify para CH31/32 e deixe tocar 5 segundos AX_CAPTURE.summary() AX_CAPTURE.export()

Cada comando mostra logs em verde (TX →) e magenta (← RX) no console.

Controles Disponíveis

AX_CAPTURE.clear() // Limpa logs AX_CAPTURE.toggle() // Ativar/desativar AX_CAPTURE.summary() // Resumo de TX/RX AX_CAPTURE.export() // Salva JSON para arquivo AX_CAPTURE.sequence() // Sequência de eventos AX_CAPTURE.logs // Array bruto de logs

4. Use os comandos de teste acima
